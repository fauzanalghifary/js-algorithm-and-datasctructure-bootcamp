## Object

- When you don't need order
- When you need fast access / insertion and removal

- Insertion O(1)
- Removal O(1)
- Searching O(n)
- Access O(1)

- Object.keys O(n)
- Object.values O(n)
- Object.entries O(n)
- hasOwnProperty O(1)

## Arrays

- When you need order
- When you need fast access / insertion and removal (sort of ...)

- Insertion (depends)
- Removal (depends)
- Searching O(n)
- Access O(1)

- push O(1)
- pop O(1)
- shift O(n)
- unshift O(n)
- concat O(n)
- slice O(n)
- splice O(n)
- sort O(n log n)
- forEach/map/filter/reduce O(n)
